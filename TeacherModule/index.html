Module #2: Loops

Prompt: Create a multi-color pattern of one shape that fills up the canvas.

There are multiple types of ways to loop in Processing. One way is to use a for loop. The setup for this is for(var i = 0; i < 10; i++). The first part initializes the variable i to a value of 0. The second part is a conditional, where every time it loops it will check if this value is true and exit the loop if not. The last statement occurs every time the loops is done and will increment the variable i. The loop will run 10 times.
In order to make a pattern in Processing, you could combine loops with shapes like rect() or ellipse().
